# WishlistWatcher Chrome Extension

Track product prices directly from your browser and get alerted when they drop.

## Install (Developer Mode)

1. Clone this repo:
```
   git clone https://github.com/behnammhp/wishlistwatcher-extension.git
```
2. Open Chrome → `chrome://extensions/`
3. Enable **Developer mode** (top right toggle)
4. Click **Load unpacked** → select the cloned folder

## Update
```
git pull
```
Then click the 🔄 refresh icon on the extension card in `chrome://extensions/`.

## Features
- Click extension icon on any product page → auto-fetches price → set target → track
- Right-click any product link → "Track this product 🏷️"
- Shows original vs sale price with discount %
- Login with your WishlistWatcher account
- Auto-login when opening full dashboard
